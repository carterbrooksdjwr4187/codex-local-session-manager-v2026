## Overview

Codex Local Session Manager is a compact browser utility for inspecting and maintaining Codex session history on your local machine. It ships as one HTML file, so you can open it straight in a browser and manage session data without spinning up a server or relying on a backend service.

It is aimed at anyone who wants a quicker way to review lengthy conversation logs, locate specific sessions, and clear away older entries. By putting browsing, previewing, and cleanup tools together in one interface, it supports everyday session housekeeping with less friction.

## Features

- View Codex session history locally in the browser
- Search sessions to locate specific conversations more quickly
- Narrow the list with filters
- Move through larger histories with pagination
- Preview the full conversation text for each session
- Remove the active session, chosen sessions, or all sessions at once
- Delivered as a single-file HTML tool with no backend dependency
- Built around local review and cleanup workflows for sessions

## FAQ

### Does it need an internet connection?
No backend is needed. The tool is intended to run in the browser from a local file.

### How do I update it?
Download the latest build and replace your current file with the new version.

### Why am I not seeing the sessions I expect?
Review your search terms, filters, and pagination state. With large histories, the session may be on a different page.

### Can I restore deleted sessions?
Deletion actions remove entries from the local session view, so check carefully before clearing data.

### Where can I change settings?
If your build includes settings, they are generally available in the interface or stored locally in the browser context.
